# Figures & Tables Add-on

This small pack adds a single script `build_artifacts.py` that:
1) Reads a labeled CSV (e.g., generated by your `data_gen.py`: columns: `message,label`).
2) Runs your **existing** classifier from `classify.py`.
3) Saves publishable **tables (CSV)** and **figures (PNG)** for your paper:
   - `artifacts/tables/metrics_overall.csv`
   - `artifacts/tables/metrics_by_class.csv`
   - `artifacts/tables/confusion_matrix.csv`
   - `artifacts/figures/confusion_matrix.png`
   - `artifacts/figures/class_support.png`
   - `artifacts/data/predictions.csv`

## Install (only one new dependency)
```bash
# from your project folder:
pip install matplotlib
```

## Run
```bash
# Example: use the dataset produced by data_gen.py
python data_gen.py --out sample_logs.csv --n 300

# Build artifacts
python build_artifacts.py --data sample_logs.csv --outdir artifacts
```

## Notes
- Figures use Matplotlib defaults (no custom colors) and save as PNG.
